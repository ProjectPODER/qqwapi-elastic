2020-08-26: v3.0.0-beta
- Completely rewritten API. 
- New specification written in OpenAPIv3 JSON standard.
- Switched DB engine to ElasticSearch
- New endpoints: areas, search, summaries, record
- All parameters are now specified
- Breaking changes in /contracts response: it no longer returns aggregated records, but single contracts in a format based in OCDS but without the OCDS wrapping. OCDS still available in /record endpoint
- Breaking changes in page count: If there's more than 10.000 results the size is estimated. This is indicated by the new count_pressicion property, present in every response.
